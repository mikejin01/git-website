Step by Step Instruction to Start a Git Project

Step 1

Open up terminal and use the cd command to get to the directory of the project

Step 2

Run "git init" command to initialize the project

Step 3

Now you need to stage your files for the commit. To stage every file, simply use command "git add -A"

Step 4

Now you can make the commit by using the command, for example, "git commit -m "initial commit""
